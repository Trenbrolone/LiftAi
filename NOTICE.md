### What was used and how

- **Images:** one photo per exercise (the first image in each exercise's
  image set) was downloaded from the project and stored locally in
  `images/` as `<Exercise_Id>.jpg`. The images are **not** hotlinked; they
  are served from this repository so nothing depends on an external host.
- **Descriptions:** the how-to text shown in each exercise's detail view is
  the project's original public-domain `instructions` text, with each
  exercise's instruction steps joined into a single paragraph. The text is
  unmodified apart from that joining.

The selection of 114 exercises, the mapping of each exercise to one of the
app's six muscle groups (Chest, Back, Legs, Shoulders, Arms, Core), the
equipment labels, and all of the application code are original to this
project.
